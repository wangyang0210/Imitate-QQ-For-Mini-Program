# mp-color-qq

使用mpvue+colorui+vant-weapp+gateaway实现QQ的聊天功能


# 使用

克隆

```
git clone  -b mpvue-color-qq git@github.com:wangyang0210/Imitate-QQ-For-Mini-Program.git
```


安装

```
npm install
```


编译

```
npm run build
```


>打开微信开发者工具,将`dist/wx`导入进去即可
